# City360
Visualizing Multimodal City Events for Decision Support
## Front-end
- HTML
- jQuery
- Materialize CSS Framework
- Mapbox.js & Leaflet.js
## Back-end
- REST APIs (Inrix, and Eventful)
- MySQL (Open311) - Data crawling
